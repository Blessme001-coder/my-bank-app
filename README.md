# 🏦 THE BANK OF ODOTEH

A desktop banking application built with Python and Tkinter.

## Features
- Create and manage bank accounts
- Deposit and withdraw funds
- View transaction history
- Dark theme UI
- Data saved automatically to JSON

## How to run
1. Make sure Python is installed
2. Clone this repo
3. Run: py ui.py

## Built with
- Python 3.13
- Tkinter
- JSON file storage
